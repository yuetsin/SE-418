# Word Ladder Project
> Based on Maven, SpringBoot, and JUnit

## Pack Project
``` shell
./mvnw package
```

## Clean and Pack Project
``` shell
./mvnw clean package
```

## Compile Project
``` shell
./mvnw compile
```
## Compile to Docker Image
``` shell
./mvnw install dockerfile:build
```

## Clean Project
``` shell
./mvnw clean
```

## Run Test
``` shell
./mvnw test
```
