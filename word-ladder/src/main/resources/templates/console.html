Here's word ladder console...

GET /auditevents，显示应用暴露的审计事件

GET /beans，应用程序上下文中的 Bean

GET /conditions，显示应用的自动配置条件

GET /configprops，描述配置属性注入 Bean 情况

GET /env，获取环境属性

GET /flyway，获取 Flyway 数据库迁移信息

GET /liquidbase，获取 Liquibase 数据库迁移信息

GET /health，报告应用程序的健康指标

GET /heapdump，dump 一份 JVM 堆

GET /httptrace，显示 HTTP 足迹

GET /info，获取应用的定制信息

GET /logfile，返回 logfile 中的内容

GET /loggers，显示和修改配置的 loggers

GET /metrics，报告各种度量信息

GET /scheduledtasks，报告应用中的定时任务

GET /sessions，显示应用中的 HTTP sessions 信息

GET /mappings，描述全部的 URI 路径，以及它们的控制器

GET /threaddump，获取线程活动的快照
